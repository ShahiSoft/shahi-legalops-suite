# Contributing to ShahiTemplate

We appreciate contributions! Please follow these guidelines to maintain quality and consistency.

## Getting Started
- Fork the repository
- Create a feature branch
- Follow WordPress Coding Standards
- Add tests where practical

## Development
- Use prepared statements for all DB queries
- Sanitize all inputs and escape all outputs
- Use nonces for forms and AJAX
- Highlight all `PLACEHOLDER:` markers and mock data in code

## Code Quality
- Run:
  - `composer sniff`
  - `composer analyse`
  - `composer test`

## Commits & PRs
- Write clear commit messages
- Submit focused PRs
- Fill PR template thoroughly

## Reporting Issues
- Use the Issue Templates
- Include environment details and steps to reproduce

Thank you for helping improve ShahiTemplate!
