## üìÅ Module Overview

### Purpose
The Accessibility Scanner Module provides comprehensive WCAG 2.1/2.2 compliance scanning and reporting for WordPress sites.

### Key Features
- Automated accessibility scanning
- WCAG 2.1/2.2 compliance checking
- Detailed violation reporting
- Remediation guidance
- Dashboard integration
- Real-time monitoring

### Compliance Standards
- **WCAG 2.1** - Web Content Accessibility Guidelines 2.1
- **WCAG 2.2** - Web Content Accessibility Guidelines 2.2
- **Section 508** - U.S. federal accessibility requirements
- **ADA** - Americans with Disabilities Act compliance
